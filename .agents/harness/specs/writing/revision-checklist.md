# Revision Checklist

Apply in order. Each pass focuses on ONE dimension.

## Pass 1: Structure
- Does each section have a clear purpose?
- Is there a logical progression (not just association)?
- Could a section be removed without loss? If yes, remove it.
- Does the opening earn the reader's attention? (No throat-clearing)

## Pass 2: Advancement
- Does every sentence say something new?
- Mark any sentence that rephrases the previous one → delete it
- Mark any paragraph that circles back → restructure or cut

## Pass 3: Concreteness
- Abstract noun clusters ("the nature of the complexity") → replace with specific images
- Vague verbs ("seems to suggest") → commit to a claim or cut
- Generic examples → replace with specific, surprising ones

## Pass 4: Rhythm
- Vary sentence length (mix short declarative with longer complex)
- No three consecutive sentences starting with the same word
- Read aloud: does it flow? Where do you stumble?

## Pass 5: Cut 20%
- After all other passes, enforce a 20% word count reduction
- Target: filler words, redundant qualifiers, hedging language
- Every word must earn its place

## Quality Gate
- Would Alain de Botton publish this? (clarity + insight + accessibility)
- Does each paragraph contain at least one genuine insight?
- Is there anything you're keeping just because you wrote it? (sunk cost → cut)
